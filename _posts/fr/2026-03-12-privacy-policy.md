# Privacy Policy

**Last Updated:** March 12, 2026

This Privacy Policy describes how **stepan** ("we", "our", or "us") collects, uses, and protects information when users interact with the **stepan** application.

The **stepan** application is operated by **stepan** and available through the website:

https://stepan.be

The application is intended for **educational purposes** and integrates with the TikTok API.

---

## 1. Information We Collect

When users connect their TikTok account to the stepan application through TikTok's authorization system, we may collect certain information permitted by the TikTok API, including:

- TikTok user ID
- Username
- Public profile information
- Account metadata provided by TikTok
- Access tokens required to access the TikTok API

We **do not collect TikTok passwords**.

---

## 2. How We Use Information

We use collected information only for the following purposes:

- To operate and provide the stepan application
- To authenticate users via TikTok
- To access TikTok data through the official TikTok API
- To support educational demonstrations and development
- To maintain and improve the application

We **do not sell, rent, or trade user data**.

---

## 3. Data Storage and Security

We take reasonable technical and organizational measures to protect user information from unauthorized access, misuse, or disclosure.

Any access tokens or identifiers obtained through TikTok are stored securely and used only for providing the application's functionality.

---

## 4. Sharing of Information

We do not share personal information with third parties except:

- When required by law
- When required to comply with TikTok Developer policies
- When necessary for secure hosting or infrastructure providers

---

## 5. Data Retention

User information is retained only as long as necessary to operate the service or until a user requests deletion.
